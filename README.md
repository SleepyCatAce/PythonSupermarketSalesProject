# PythonSupermarketSalesProject
The Final Project for my Data Handling and Visualisation Subject

This project had to follow a couple of principles. They were
- Finding and reading a dataset with at least 100 rows
- Showing the first 5 lines of the dataset
- Displaying 5 different types of graphs
- The correlation between all of the variables - both in matrix and heatmap form

This was done in Python in Jupyter Notebook using multiple libaries, including numpy, pandas, matplotlib.pyplot, seaborn, datetime and scipy.stats

The dataset used can be found here: https://www.kaggle.com/datasets/aungpyaeap/supermarket-sales
